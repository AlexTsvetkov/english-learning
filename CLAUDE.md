# CLAUDE.md — Project Instructions for AI Assistants

## Project Overview

This is a **12-week B2→C1 English improvement course** for Russian speakers, themed around psychology, mental health, emotions, and personal growth. The repository contains structured learning materials — not source code.

## Repository Structure

```
grammar/          — Weekly grammar modules (~45 min each)
vocabulary/       — Weekly vocabulary modules (~30 min each)
essay/            — Weekly writing assignments (~45 min each)
COURSE_PROMPT.md  — Full AI tutor prompt used to generate course content
README.md         — Course syllabus, schedule, and study guide
CHANGELOG.md      — Version history
```

### File Naming Convention

All weekly files follow the pattern:
```
week-{NN}-{Theme-Name}.md
```
Examples: `week-01-Understanding-Emotions.md`, `week-12-Integration-Personal-Growth.md`

Special files:
- `grammar/mid-course-check.md` — Diagnostic quiz after Week 6
- `essay/final-project.md` — Final 500–700 word personal essay assignment

## Content Guidelines

When editing or creating course materials:

- **Target audience**: Russian-speaking adults at B2 level progressing to C1
- **Tone**: Warm, encouraging, intellectually stimulating — not dry or textbook-like
- **Grammar modules** include explanations, Russian comparisons, common mistakes, 5–8 exercises, and a contextual writing task
- **Vocabulary modules** include 20–25 target words with definitions, examples, Russian translations, collocations, register notes, phrasal verbs, idioms, and practice exercises
- **Writing modules** include a model text (200–300 words), analysis, a writing technique lesson, and a writing task (250–350 words)
- All three modules in a given week share the same thematic focus
- Use Markdown formatting: headings, tables for word lists, numbered exercises
